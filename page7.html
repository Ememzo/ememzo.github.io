OUR AWESOME TEAM




BTS Foundation organisation is comprised of members who have been involved individually- as ACW Volunteers - in delivering ICT skills and ICT knowledge to the youths in Edo state. They have been doing this in collaboration with the Edo State ICT Agency, the Edo State Education Board and the Africa Code Week ( ACW ). ACW is an initiative of the Tech giant company SAP aimed at providing ICT and coding skills and knowledge to youths in Africa. In 2018 the members were involved as ACW volunteers in running the coding program in Edo State for the first time that reached over 100 thousand youths 


In 2019 the members were again involved as ACW volunteers, collaborating with the Edo State ICT Agency and the Edo State Education Board to run the coding program to youths that reached over 200 thousand youths – link to the 2019 ACW run in Edo state.
